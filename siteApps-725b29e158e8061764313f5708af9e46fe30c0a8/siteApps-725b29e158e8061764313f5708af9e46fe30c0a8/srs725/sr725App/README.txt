
Stanford Research SR 725

Rb Frequency Standard

http://www.thinksrs.com/products/FS725.htm

Written and tested with firmware 3.24

= Coverage

Impliments most monitoring and some control points.
Most control points also support save to EEPROM.

= Dependencies

EPICS Base, Asyn, and Stream Device

Tested with
Base 3.14.10
Asyn 4.10
Stream Device 2.4

= Usage

The example ioc 'iocsr725' demonstrates use with a terminal server which
support raw sockets mode (not reverse telnet).

When connecting directly use 9600 8N1 with no flow control.
